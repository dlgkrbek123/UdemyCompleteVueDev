Vue instance 생성

Vue.createApp({}).mount('#app');
=> 데이터 관리 + DOM 조작
=> 옵션 전달 가능
=> 마운트 : 문서에 Vue를 삽입하는 작업
=> 위젯을 써야하는 경우가 아니면 보통 한페이지에 인스턴스는 하나만 사용

=> Vue에 의해 선택될 엘리먼트에 적용
