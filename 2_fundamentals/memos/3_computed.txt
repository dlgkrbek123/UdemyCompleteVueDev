computed
=> 의존성이 변경되었을 때만 값을 리턴하는 함수를 제공
=> 의존성을 자동으로 계산
=> 값으로서 처리 (useMemo와 유사)

-------------------------------------------------------------------------------------------

watchers
=> data 변동을 감시
=> useEffect와 유사

watch: {
  age(newVal, oldVal) {
    setTimeout(()=> {
      this.age = 20
    }, 3000)
  },
},
