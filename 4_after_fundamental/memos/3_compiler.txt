템플릿은 두가지 방식으로 생성 가능
1. html파일에 작성
2. 인스턴스에 template 프로퍼티 추가
=> 권장안하는 이유
  1. 에디터에서 js내에 html 문법에 대한 처리가 미비
  2. 멀티라인 템플릿 작성이 쉽지 않다

뷰는 컴파일러가 있는 버전과 없는 버전으로 갈린다
뷰 내부적으로 Vue.compile()을 호출해 템플릿을 처리한다.


vue 인스턴스에 render(){} 를 오버라이딩 가능
React.createElement와 유사
컴파일러 없이 사용

render() {
  return Vue.h(
    'h1',
    this.message
  );
}
